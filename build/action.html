Some Features

Browserify
Use commonjs and Require npm modules in your app for your browser!
Livereload
Make a change to any of the ./app files and livereload will take care of the rest!
Any javascript file will trigger livereload, but only index.html and app.css will livereload. Use a build system to generate your app.css (like sass or less.
Front-end Framework / library agnostic
Does not care about what library you use.
Currently uses jquery as the only dependency, but you can remove it!
The convention is, front-end modules are dependencies, while back-end modules are devDependencies
